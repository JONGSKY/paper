## LeNet-5이란
  - LeNet은 우편번호와 수표의 필기체를 인식하기 위해 개발되었습니다. CNN 모델을 최초로 개발한 사람은 프랑스 출신의 Yann LeCun이며, 1989년 “Backpropagation applied to handwritten zip code recognition” 논문을 통해 최초로 CNN을 사용하였고, 이후 1998년 LeNet이라는 Network를 소개하였습니다.

#### LeNet-5의 구조

![lenet5_구조](https://user-images.githubusercontent.com/40276516/74128881-98d5c680-4c21-11ea-9d33-cdde723ea337.png)

  - LeNet5는 총 7개의 Layer로 구성되어있습니다. 두개의 Convoltuion Layer, 2개의 Sub-Sampling Layer, 2개의 Fully-Connected Layer 그리고 최종 출력 Layer로 이루어져 있습니다.
